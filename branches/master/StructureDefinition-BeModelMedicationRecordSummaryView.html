Logical Model: Medication Record Logical Model - Medication Summary View detail

Official URL: http://hl7belgium.org/fhir/riziv-medicationrecord/StructureDefinition/BeModelMedicationRecordSummaryView Version: 0.1.0-test
Active as of 2023-02-03 Responsible: HL7 Belgium Realm: Computable Name: BeModelMedicationRecordSummaryView

A model for representing a summary view of a patient’s medication.

Usage:

  • This Logical Model Profile is not used by any profiles in this Implementation Guide
